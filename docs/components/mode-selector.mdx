# Mode selector

The mode selector controls the [pool filtration mode](https://docs.page/mguyard/hass-iopool/integration/entities#pool-mode-selector-advanced).

- It reads the pool mode select entity.
- It displays the current mode and three selectable options.
- Tapping an option changes the filtration mode through Home Assistant.
