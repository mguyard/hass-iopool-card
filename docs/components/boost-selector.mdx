The boost selector shows the active boost state and lets you start or stop a boost.

- It reads the [boost select entity](https://docs.page/mguyard/hass-iopool/integration/entities#boost-selector-advanced) and its start and end timestamps.
- The active option is highlighted.
- A countdown bar appears while the boost is active.
