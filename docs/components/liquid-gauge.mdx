# Liquid gauge

The liquid gauge renders the temperature, pH, and ORP values as animated circular gauges.

- It reads the sensor entity for the current measure.
- It uses the configured thresholds to choose red, orange, or green.
- It shows the measure value and a short status label above the gauge.
