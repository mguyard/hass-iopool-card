# Filtration progress

This section shows the elapsed daily filtration time, the target duration, and the iopool recommendation.

- It reads the filtration binary sensor and elapsed filtration sensor.
- It uses a progress bar to show how far through the daily target the pool is.
- When the target is exceeded, the bar stays full and the percentage remains visible.
