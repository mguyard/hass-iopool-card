# Installation

<Info>
  There is no separate HACS repository for the card. Install the hass-iopool integration through
  HACS, and the card is exposed automatically.
</Info>

## Install hass-iopool

1. Open HACS in Home Assistant.
2. [Install the hass-iopool integration.](https://docs.page/mguyard/hass-iopool/integration)
3. Restart Home Assistant if prompted.

## Add the card

1. Open a dashboard in edit mode.
2. Add a new card.
3. Search for iopool Card.
4. Select the card and choose your pool device.

### Minimal YAML

```yaml
type: custom:iopool-card
device_id: YOUR_DEVICE_ID
```

<Warning>
  If the card does not appear immediately, refresh the browser or clear the cache once after the
  integration has been installed.
</Warning>
