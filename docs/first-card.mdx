# First card

<Steps>
  <Step title="Open dashboard edit mode">
    Open the dashboard where you want to show the pool and click Edit dashboard.
  </Step>
  <Step title="Add a new card">
    Search for iopool Card in the card picker and open the visual editor.
  </Step>
  <Step title="Select your device">Choose the iopool device for the pool you want to display.</Step>
  <Step title="Save the card">Save the card and exit edit mode.</Step>
</Steps>

## Minimal YAML

```yaml
type: custom:iopool-card
device_id: YOUR_DEVICE_ID // [!code highlight]
```
