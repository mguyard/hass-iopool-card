# Debug mode

Enable debug mode in YAML only:

```yaml
debug: true
```

When enabled, the card prints lifecycle and data-resolution details in the browser console.

<Info>
  Debug mode is useful when verifying device resolution, chart loading, or action behavior.
</Info>
