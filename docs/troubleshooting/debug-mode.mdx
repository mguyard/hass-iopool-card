# Debug mode

Enable debug mode in YAML only:

```yaml
debug: true // [!code highlight]
```

When enabled, the card logs device resolution, the resolved entity map, chart loading, and section action execution in the browser console.

Debug mode is not available in the visual editor, so it must be set in YAML.

<Info>
  Open browser DevTools with F12, then switch to the Console tab to inspect the logs.
</Info>

Turn debug mode off after troubleshooting. Leaving it enabled can add noise to the console and slightly affect performance.
