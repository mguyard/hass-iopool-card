Temperature thresholds define the four transition values that create the five gauge zones:
red-low, yellow-low, green, yellow-high, and red-high.

```yaml
temperature_thresholds: [15, 20, 29, 32] // [!code highlight]
```

The default pool preset is `[15, 20.5, 29, 32]`. A spa-friendly preset commonly uses `[35, 37, 39, 40]`.

<Info>Only temperature thresholds can be customized.</Info>

Keep the values strictly increasing. If two adjacent values are equal or out of order, the editor rejects the change.
