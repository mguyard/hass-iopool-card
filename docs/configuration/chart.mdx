# Chart

The temperature chart is enabled by default and can be disabled with `show_chart: false`.

```yaml
show_chart: true // [!code highlight]
chart_period: 96 // [!code highlight]
```

The period selector supports four values: `24` for 24h, `48` for 2d, `96` for 4d, and `168` for 7d.

<Info>
  Changing the period updates the chart window immediately and refreshes the history data for that
  range.
</Info>
